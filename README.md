#  Spam Email Detection System

A Machine Learning application built with Python to detect spam emails using Natural Language Processing (NLP).

## Features
- Uses **Naive Bayes** and **Logistic Regression** classifiers.
- Text preprocessing using regex and stopword removal.
- Vectorization using **TF-IDF**.
- Evaluation metrics: **Accuracy, Precision, Recall, F1-Score**.
- Interactive **Streamlit** web dashboard.
